# HRD Research Reference Checklist

Use this checklist when checking reference lists, in-text/reference matching, or APA 7 reference formatting.

## Citation Coverage

- Does every in-text citation appear in the reference list?
- Does every reference-list entry appear in the manuscript?
- Are author spellings consistent?
- Are publication years consistent?
- Are same-author same-year works labeled with `a`, `b`, `c` where needed?

## Reference Entry Completeness

- Journal article: authors, year, article title, journal title, volume, issue if supplied, pages/article number, DOI if available.
- Book: authors, year, title, publisher, DOI if available.
- Edited chapter: chapter authors, year, chapter title, editors, book title, page range, publisher, DOI if available.
- Report/webpage: organization or author, year/date, title, publisher or site, URL.

## APA 7 Formatting

- Article and chapter titles use sentence case.
- Journal and book titles use title case.
- Journal volume is included when supplied.
- DOI is in URL format when supplied.
- References are alphabetized.
- Duplicate or near-duplicate entries are flagged.

## Integrity Rules

- Do not invent missing details.
- Mark uncertain details with bracketed notes.
- Separate formatting fixes from source-verification needs.
- If only a reference list is provided, do not claim in-text/reference matching was checked.
