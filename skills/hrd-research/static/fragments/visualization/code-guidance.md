# Visualization: Code Guidance

Use when the user asks for R, Python, Excel, SPSS, Stata, or other chart-making steps.

Ask for or infer only when available:

- data format
- variable names
- intended chart type
- grouping variables
- output format
- journal or APA formatting needs

Prefer reproducible code when the user provides data or variable names. Use placeholders only when needed, and label them clearly.

For R, prefer `ggplot2` style logic. For Python, prefer `pandas` plus `matplotlib`, `seaborn`, or `plotly` depending on static or interactive needs.
