# Visualization: APA Table

For APA-style tables:

- Use a clear table number and italicized title in final prose when possible.
- Put comparable values in columns readers can scan.
- Align decimals consistently.
- Define abbreviations, scales, and significance markers in notes.
- Avoid vertical lines and excessive borders.
- Use enough precision for interpretation, not more.

Good HRD research tables often clarify sample, context, construct measurement, descriptive statistics, correlations, model estimates, or qualitative theme evidence.
