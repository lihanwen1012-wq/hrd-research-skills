# Task: Interpret Visual

Use when the user provides a chart, table, model figure, or figure description and asks for explanation.

Interpret in this order:

1. What the visual shows descriptively.
2. What comparison, trend, distribution, or relationship matters.
3. What the visual suggests for the HRD research question.
4. What cannot be concluded from the visual alone.

Avoid causal language unless the design supports it.
