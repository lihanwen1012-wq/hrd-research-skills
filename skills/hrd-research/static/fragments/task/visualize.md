# Task: Visualize

Use when the user asks to create, choose, revise, or specify a graph or chart.

First identify:

- research question or claim the visual should support
- unit of analysis
- variable types
- comparison groups
- time or sequence, if any
- uncertainty or distribution information
- target outlet or APA expectation

Prefer visuals that make the HRD research claim easier to evaluate, not merely more decorative.

Do not invent data, labels, units, p values, confidence intervals, or effect sizes.
