# Citation: Citation-Reference Match

Use when the user provides manuscript text plus a reference list, or asks whether citations and references match.

Check both directions:

1. In-text citation to reference list: every cited author-year pair should have a matching reference entry.
2. Reference list to in-text citation: every reference entry should be cited in the manuscript or marked as uncited.

Flag:

- missing reference entry
- uncited reference entry
- author spelling mismatch
- year mismatch
- multiple same-author same-year entries needing `a`, `b`, `c`
- institutional author mismatch
- ambiguous shortened citation such as `et al.` when multiple references could match
