# Citation: APA Reference Format

Use when formatting or checking APA 7 reference entries.

General APA 7 reminders:

- Alphabetize by first author surname.
- Use initials for given names.
- Use sentence case for article, chapter, and report titles.
- Use title case and italics for journal titles and book titles in final formatted prose when possible.
- Include journal volume in italics, issue in parentheses, and page range or article number when supplied.
- Use DOI URLs as `https://doi.org/...` when DOI is provided.
- Use hanging indent in final document formatting, but do not simulate indentation when plain text is requested unless helpful.

Do not invent missing fields. Use bracketed notes for unresolved details.

Common entry patterns:

- Journal article: Author, A. A., & Author, B. B. (Year). Article title. Journal Title, volume(issue), page-page. https://doi.org/...
- Book: Author, A. A. (Year). Book title. Publisher.
- Edited chapter: Author, A. A. (Year). Chapter title. In E. E. Editor (Ed.), Book title (pp. xx-xx). Publisher.
- Webpage/report: Author or Organization. (Year). Title. Site or Publisher. URL
