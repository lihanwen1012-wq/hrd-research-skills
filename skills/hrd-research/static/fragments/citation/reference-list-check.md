# Citation: Reference List Check

Use when reviewing a reference list or bibliography.

Check each reference entry for:

- author names and order
- publication year
- title
- source title
- journal volume, issue, and page or article number when applicable
- publisher for books
- editors and chapter page range for edited book chapters
- DOI or URL when supplied or expected
- likely duplicates
- APA 7 capitalization and punctuation

Do not fabricate missing details. Mark missing or uncertain parts clearly.
